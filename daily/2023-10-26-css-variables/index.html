Mastering CSS Variables

Oct 26, 2023 In: frontend

CSS Variables (Custom Properties) make it so easy to implement dark mode.

:root {
  --bg-color: #ffffff;
  --text-color: #333333;
}

body.dark-mode {
  --bg-color: #121212;
  --text-color: #e0e0e0;
}

By toggling a class on the body, the entire theme changes instantly!
