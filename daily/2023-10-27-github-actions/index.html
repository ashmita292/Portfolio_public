Automating with GitHub Actions

Oct 27, 2023 In: devops

I set up a GitHub Actions workflow to build and deploy my site to GitHub Pages whenever I push to main.

It uses the peaceiris/actions-gh-pages action, which is super convenient. Now I don’t have to manually build the site every time.
